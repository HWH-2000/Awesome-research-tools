# Awesome-research-tools

## Arxiv tool
将Arxiv网网址中 arxiv -> ar5iv 可以得到arxiv论文html5格式，再使用google translate全文翻译   
like this：https://arxiv.org/abs/2010.11929    ->   https://ar5iv.org/abs/2010.11929
